sitiosbo

Sitios web y landing pages profesionales para negocios en Bolivia — sin costo de hosting ni dominio para el cliente.

🔗 Sitio en vivo: sitiosbo.github.io

¿Qué es esto?

sitiosbo diseña y publica landing pages para negocios locales (clínicas, comercios, profesionales independientes) que no tienen presencia web o cuya página anterior dejó de funcionar por falta de pago de hosting/dominio.

Cada cliente recibe:


Landing page profesional, adaptada a celular
Hosting incluido sin costo recurrente
Formulario de contacto conectado a Google Sheets
SEO básico para aparecer en búsquedas locales


Estructura del proyecto

Cada cliente vive en su propio repositorio dentro de esta organización, publicado en:

sitiosbo.github.io/nombre-del-cliente

Este repositorio (sitiosbo.github.io) es la página raíz / portafolio del servicio.

Stack técnico


HTML / CSS / JS estático
Hosting: GitHub Pages
Formularios: Google Apps Script → Google Sheets
Sin frameworks, sin dependencias de pago
